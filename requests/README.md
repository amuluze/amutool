## requests

类似 Python requests 的库的 http 客户端工具。

```shell
GET: url params headers cookies 
POST: url data json headers cookies
  - data: 要发送到指定 url 的 map slice byte 或 file
  - json: 要发送到指定 url 的 json 对象
```
